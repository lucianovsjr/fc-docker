O docker foi feito para rodar no Linux.

O Docker Desktop emula uma máquina virtual, o Docker host é a máquina virtual não a própria máquina.

Esse tipo de network funciona de forma correta somente no Linux e Window com WSL2

```bash
>> docker run --rm -d --name fc-nginx --network host nginx
>> curl http://localhost
```
